# 🚀 SocialNexus - Painel SMM Premium

Painel de revenda de serviços de redes sociais (SMM) totalmente automatizado e integrado com a API da **GrowFollows**.

## ✨ Principais Recursos
- **+2.300 Serviços:** Integração total com categorias reais (Instagram, TikTok, YouTube, etc).
- **Tradução Automática:** Painel 100% em Português (Brasil).
- **Gestão de Lucro:** Ajuste de margem global ou individual em tempo real.
- **Hub de Pagamentos:** Suporte para Pix, Cartão e Criptomoedas.
- **Visual Premium:** Design moderno com modo escuro, animações e responsividade total (Mobile-First).

## 🛠️ Tecnologias Utilizadas
- HTML5 / CSS3 (Vanilla)
- JavaScript (ES6+)
- Firebase (Sincronização de dados)
- API GrowFollows

## ⚙️ Configuração
Para configurar sua conta, ajuste as credenciais no arquivo `app.js` e `firebase-sync.js`.
